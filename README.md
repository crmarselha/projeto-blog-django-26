## 🚀 Como executar o projeto

Este projeto utiliza **Docker** para facilitar o ambiente de desenvolvimento.

1. Certifique-se de ter o Docker instalado.
2. Clone o repositório.
3. Configure os arquivos na pasta `dotenv_files/` (use o `.env-example` como base).
4. Execute o comando:
   ```bash
   docker-compose up --build
